# CRUD-Web-App-Laravel
Web app created with PHP Laravel that supports CRUD operations (Create, Read, Update, Delete) to a clinic database.

First you need to have PHP and Laravel installed:
Start cmd and use the following commands:

cd path\to\folder\
php artisan serve

Then hold Ctrl and click on the http link provided and the app will open in your browser.

The interface is simple, I focused on the backend part more and I did not separated the admin from the usual user.
